|  参数  |  描述  |  类型  |  默认值  |  可选值  |

|  id  |  图标 id  |  String  |  --  |  如上表所示  |

|  color  |  图标颜色  |  String  |  #000  |  --  |
|  rotate  |  图标顺时针旋转的角度值  |  Number  |  0  |  --  |
