# Tag 标签

## Tag Snippets

### 基础用法

![](img/tag/tag.png)

```html
<fm-tag color="#f12528">推荐</fm-tag>
```

### 小标签

![](img/tag/tag-small.png)

```html
<fm-tag color="#f12528" size="small">热映</fm-tag>
```

## Tag Attributes

|  参数  |  描述  |  类型  |  默认值  | 可选值  |
|  -----  |  -----  |  -----  |  -----  |  -----  |
|  size  |  标签显示尺寸  |  String  |  --  |  small  |
|  color  |  标签颜色  |  String  |  #198ded  |  --  |
