# Toast 提示

## Toast Snippets

### 基础用法

![](/doc/img/toast/toast.png)

```javascript
this.$toast('Flyme UI');
```

### 状态提示

![](/doc/img/toast/toast-success.png)

```javascript
this.$toast('发送成功', 'success');
```

## Toast Parameters

|  参数  |  描述  |  类型  |  默认值  | 可选值  |
|  -----  |  -----  |  -----  |  -----  |  -----  |
|  content  |  提示的文本内容  |  String  |  --  |  --  |
|  type  |  提示的类型  |  String  |  --  |  success  |
