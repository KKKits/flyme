# Flyme UI

[![npm package](https://img.shields.io/npm/v/flyme-ui.svg)](https://www.npmjs.org/package/flyme-ui)

> 一套基于 Vue.js 的移动端 UI 组件库

## 使用文档

[中文文档](https://fedesigner.gitbooks.io/flyme/content/doc/install.html)

## 开发

安装依赖：

```
npm install
```

运行 Demo：

```
npm test
```

在浏览器中访问：

```
http://localhost:8080/
```
